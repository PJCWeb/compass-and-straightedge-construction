# 一个好用的尺规作图工具
一个基于HTML5 Canvas的尺规作图模拟器，支持精确的几何作图操作。
-------------
## 1.✨ 功能特性
### （1）基础工具
| 工具 | 说明 |
|------|------|
| **点工具** | 在画布上添加点 |
| **线段工具** | 绘制两点之间的线段 |
| **直线工具** | 绘制无限延伸的直线 |
| **圆工具** | 通过圆心和半径画圆 |
| **交点工具** | 计算两个图形的交点 |

### （2）核心功能
- 🎯 **智能点吸附** - 鼠标靠近已有点时自动吸附，确保精确作图
- 📜 **历史记录** - 支持撤销/重做操作
- 🔍 **画布控制** - 缩放、平移（按住Ctrl+拖拽或鼠标中键）
- 💾 **保存图片** - 将当前画布保存为PNG

### （3）操作方式
| 操作 | 方法 |
|------|------|
| **缩放** | 鼠标滚轮 或 右下角缩放按钮 |
| **平移** | Ctrl+鼠标拖拽 或 鼠标中键拖拽 |
| **画圆** | 先点圆心，再点半径点（自动吸附到已有点） |
| **交点** | 依次点击两个图形，自动计算并生成交点 |
